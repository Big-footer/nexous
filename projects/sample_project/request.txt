# 샘플 프로젝트 요청

## 작업 요청
Python을 사용하여 간단한 데이터 분석 보고서를 작성해주세요.

## 요구사항
1. 샘플 데이터 생성 (판매 데이터)
2. 기본 통계 분석 수행
3. 결과를 Markdown 형식으로 정리

## 기대 결과
- 데이터 요약 통계
- 주요 인사이트
- 시각화 추천

## 참고사항
- 이 요청은 PROMETHEUS 시스템 테스트용입니다.
- PlannerAgent가 계획을 수립하고
- ExecutorAgent가 실행하며
- WriterAgent가 보고서를 작성하고
- QAAgent가 검토합니다.
